Introduction Last updated: 2020-07-25

Using this sound editor, you'll be able to create anything with sound.

Installation

Now that you know what exactly this sound editor is, let's download and install it! First of all, please make sure you have the following on your machine:

Prerequisite Download link
Python 3(.7.4) https://www.python.org/downloads/
FFmpeg https://ffmpeg.org/download.html

Download and install

You can download the latest release of this sound editor here and then unzip the .zip file. Now navigate through a command prompt (eg. terminal, cmd.exe) to the brand new sound-editor/ folder, where you execute py -m pip install -r requirements.txt on a Windows machine, or pip3 install -r requirements.txt on Linux or Apple.

Start

Now you can start the sound editor by either starting sound-editor.bat (Windows) or sound-editor.sh (Linux or Apple). Happy editing!
